Bananas! is a browser game that is created in honor of Harambe, our one, true king.

Objective: Dodge bullets and bombs in order to eat 5 bananas and grab the baby.

Controls: Arrow Keys



Shoutout to Phaser, which is a HTML5 game framework. Kudos to JavaScript, HTML, CSS, and MAMP (local server environment). 고맙습니다 Ron선생님. R.I.P. Harambe 5/27/1999 - 5/28/2016. We miss you everyday.
